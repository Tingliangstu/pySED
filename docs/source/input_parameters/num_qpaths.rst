num_qpaths
~~~~~~~~~~

**Syntax**

.. code-block:: text

   num_qpaths = 3

**Meaning**
   Number of path segments.

**Default**
   ``None``. Set it explicitly.

**Notes**
   A path with labels ``GMKG`` has three segments: Gamma to M, M to K, and K to
   Gamma.
