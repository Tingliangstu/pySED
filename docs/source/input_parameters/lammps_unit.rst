lammps_unit
~~~~~~~~~~~

**Syntax**

.. code-block:: text

   lammps_unit = 'metal'

**Meaning**
   LAMMPS unit style used for velocity conversion.

**Default**
   ``metal``.

**Allowed values**
   ``metal`` or ``real``.

**Notes**
   ``metal`` velocities are interpreted as Angstrom/ps. ``real`` velocities are
   interpreted as Angstrom/fs.
