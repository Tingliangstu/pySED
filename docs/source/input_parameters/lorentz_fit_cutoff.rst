lorentz_fit_cutoff
~~~~~~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   lorentz_fit_cutoff = 20

**Meaning**
   Maximum frequency included in Lorentzian fitting, in THz.

**Default**
   ``None``.
