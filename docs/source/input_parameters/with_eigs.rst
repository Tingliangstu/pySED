with_eigs
~~~~~~~~~

**Syntax**

.. code-block:: text

   with_eigs = 1

**Meaning**
   Reserved for future eigenvector-related development. Normal pySED SED
   calculations do not need this parameter.

**Default**
   ``None``.

**Notes**
   This key is accepted by the parser but is not part of the current recommended
   user workflow. pySED currently uses the eigenvector-free SED expression.
