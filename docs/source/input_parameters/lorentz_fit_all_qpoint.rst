lorentz_fit_all_qpoint
~~~~~~~~~~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   lorentz_fit_all_qpoint = 1

**Meaning**
   Fit all q-points and collect frequency-lifetime data.

**Default**
   ``0``.

**Notes**
   Set ``lorentz = 1`` at the same time. Tune fitting parameters on selected
   q-points first.
