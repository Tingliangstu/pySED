time_step
~~~~~~~~~

**Syntax**

.. code-block:: text

   time_step = 1

**Meaning**
   MD time step in femtoseconds.

**Default**
   ``0``. This must be set.

**Notes**
   Use the time step from the production MD run, not from a separate relaxation
   run.
