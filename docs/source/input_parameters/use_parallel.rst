use_parallel
~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   use_parallel = 1

**Meaning**
   Enable multiprocessing over q-points.

**Default**
   ``1``.

**Notes**
   On Windows, or when memory is limited, set ``use_parallel = 0`` or use a small
   ``max_cores``.
