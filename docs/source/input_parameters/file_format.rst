file_format
~~~~~~~~~~~

**Syntax**

.. code-block:: text

   file_format = 'gpumd'

**Meaning**
   Selects the trajectory reader.

**Default**
   ``gpumd``.

**Allowed values**
   ``gpumd`` or ``lammps``.
