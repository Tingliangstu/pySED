colorbar_min
~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   colorbar_min = -20

**Meaning**
   Lower limit of the SED color scale after log scaling. Increase it to hide
   weak background noise; decrease it to reveal low-intensity branches.

**Default**
   ``None``.

**Notes**
   Run an initial plot first, then tune this value for contrast.
