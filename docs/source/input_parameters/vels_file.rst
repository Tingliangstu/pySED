vels_file
~~~~~~~~~

**Syntax**

.. code-block:: text

   vels_file = 'vels.dat'

**Meaning**
   Path to the LAMMPS velocity trajectory file. Use this only when
   ``file_format = 'lammps'``.

**Default**
   ``vels.dat``.

**Notes**
   This is used only when ``file_format = 'lammps'``.
