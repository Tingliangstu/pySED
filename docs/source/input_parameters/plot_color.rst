plot_color
~~~~~~~~~~

**Syntax**

.. code-block:: text

   plot_color = 'RdBu_r'

**Meaning**
   Matplotlib colormap for the SED intensity.

**Default**
   ``RdBu_r``.

**Examples**
   ``jet``, ``Spectral``, ``inferno``.
