plot_cutoff_freq
~~~~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   plot_cutoff_freq = 20

**Meaning**
   Maximum frequency shown in the SED map, in THz.

**Default**
   ``None``.

**Notes**
   If omitted, pySED plots the available FFT frequency range.
