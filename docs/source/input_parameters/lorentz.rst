lorentz
~~~~~~~

**Syntax**

.. code-block:: text

   lorentz = 1

**Meaning**
   Turns Lorentzian peak fitting on or off. Use it in plot/fitting mode after
   SED data already exists; pySED then fits detected peaks and can output
   phonon lifetimes.

**Default**
   ``0``.
