dump_xyz_file
~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   dump_xyz_file = '../gpumd_run/dump.xyz'

**Meaning**
   GPUMD extended XYZ trajectory file containing positions and velocities.

**Default**
   ``dump.xyz``.

**Notes**
   This is used only when ``file_format = 'gpumd'``.
