if_show_figures
~~~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   if_show_figures = 0

**Meaning**
   Show figures interactively.

**Default**
   ``0``.

**Notes**
   Use ``0`` on clusters or headless environments.
