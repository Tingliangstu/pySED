pos_file
~~~~~~~~

**Syntax**

.. code-block:: text

   pos_file = 'pos.dat'

**Meaning**
   Path to the LAMMPS position trajectory file. Use this only when
   ``file_format = 'lammps'``.

**Default**
   ``pos.dat``.

**Notes**
   This is used only when ``file_format = 'lammps'``.
