plot_SED
~~~~~~~~

**Syntax**

.. code-block:: text

   plot_SED = 0

**Meaning**
   Select compute mode or plot/fitting mode.

**Default**
   ``0``.

**Modes**
   ``0`` computes SED data from the trajectory. ``1`` reads existing SED output
   files and plots or fits them.
