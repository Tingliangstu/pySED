colorbar_max
~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   colorbar_max = -4

**Meaning**
   Upper limit of the SED color scale after log scaling. Increase it when the
   strongest branches saturate the plot; decrease it when weak branches are too
   hard to see.

**Default**
   ``None``.
