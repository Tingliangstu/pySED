plot_slice
~~~~~~~~~~

**Syntax**

.. code-block:: text

   plot_slice = 1

**Meaning**
   Plot the SED spectrum at one q-point.

**Default**
   ``0``.

**Notes**
   Use this before all-q-point Lorentz fitting to tune peak detection settings.
