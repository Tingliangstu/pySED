out_files_name
~~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   out_files_name = 'bulk_MoS2'

**Meaning**
   Prefix used for output files.

**Default**
   No parser default. Set it explicitly.

**Outputs**
   ``bulk_MoS2.SED``, ``bulk_MoS2.Qpts``, ``bulk_MoS2.THz``,
   ``bulk_MoS2.Q_distances_and_labels``, and ``bulk_MoS2-SED.png``.
