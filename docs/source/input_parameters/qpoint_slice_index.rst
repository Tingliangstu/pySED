qpoint_slice_index
~~~~~~~~~~~~~~~~~~

**Syntax**

.. code-block:: text

   qpoint_slice_index = 2

**Meaning**
   Zero-based q-point index used for single-q-point slice plotting and fitting.

**Default**
   ``0``.
